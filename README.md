# High Street Digital Website

A complete static website for High Street Digital, built with HTML5, CSS3, vanilla JavaScript, Font Awesome, Google Fonts, and AOS.

## Pages

- `index.html`
- `services.html`
- `about.html`
- `pricing.html`
- `contact.html`
- `privacy.html`
- `404.html`

## Deploying to Hostinger

Upload the full contents of this folder to your Hostinger `public_html` directory. Keep the `assets` folder beside the HTML files so images, CSS, and JavaScript load correctly.

## Customisation

Update contact details, social links, map location, prices, and service copy directly in the HTML files. The main visual style is controlled by `assets/css/style.css` and responsive refinements are in `assets/css/responsive.css`.
